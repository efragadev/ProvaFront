# Controle de Gastos

Projeto desenvolvido como MVP de uma aplicacao web de financas pessoais para a avaliacao pratica de Programacao Front-end.

## Sobre o projeto

Esta aplicacao permite registrar gastos do dia a dia de forma simples e visual. O usuario pode adicionar despesas, acompanhar o total gasto no mes, editar informacoes e remover itens da lista dinamicamente.

## Funcionalidades

- Cadastro de gastos com descricao, valor e categoria
- Listagem dinamica dos gastos na tela
- Atualizacao automatica do total gasto no mes
- Exclusao de itens com recalculo imediato do total
- Destaque visual para gastos acima de `R$ 100,00`
- Edicao de gastos ja cadastrados

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura dos arquivos

- `index.html`: estrutura da pagina
- `style.css`: estilos da aplicacao
- `main.js`: logica de interacao, renderizacao e calculos

## Objetivo da entrega

O projeto foi construido para atender aos requisitos da prova:

- formulario funcional
- listagem dinamica
- calculo automatico do total
- remocao de itens
- destaque visual para valores altos
- edicao de gastos
